# BreCaspian GitHub Pages Homepage

This is a standalone static homepage prototype. It was added without changing
the existing GitHub profile README, workflows, or generated assets.

## Preview

Open `index.html` directly in a browser.

## Deploy Later

For a real `github.io` homepage, create or use a repository named:

```text
BreCaspian.github.io
```

Then copy the contents of this directory into that repository root and push to
the `main` branch. GitHub Pages will serve it as:

```text
https://BreCaspian.github.io
```

## Style Direction

- Minimal academic homepage
- Serif typography
- Warm paper background
- Red text links
- No heavy framework or build step
