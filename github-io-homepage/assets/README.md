# Homepage Assets

Use this folder for personal homepage media.

Suggested layout:

```text
assets/
  profile/       personal portrait or avatar
  projects/      project screenshots and figures
  gifs/          demos and animated results
```

Keep file names short and URL-friendly, for example:

```text
profile/yuzhuo-yao.jpg
projects/lidar-perception-demo.png
gifs/robotics-demo.gif
```
